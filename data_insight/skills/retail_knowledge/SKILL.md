# 零售行业知识库

## 核心指标体系

### 销售类指标
- **GMV（成交总额）**：一段时间内的总销售额，包含已付款和未付款订单
- **净销售额**：GMV - 退款 - 折扣
- **客单价**：总销售额 / 成交客户数
- **连带率**：销售件数 / 成交笔数
- **坪效**：销售额 / 门店面积

### 流量类指标
- **进店客流**：进入门店的顾客数量
- **转化率**：成交客户数 / 进店客流
- **深度访问率**：进入店铺深处区域的客户比例

### 库存类指标
- **库存周转率**：销售成本 / 平均库存
- **缺货率**：缺货SKU数 / 总SKU数
- **售罄率**：已销售数量 / 进货数量

### 客户类指标
- **新客占比**：新客户数 / 总客户数
- **复购率**：重复购买客户数 / 总客户数
- **会员渗透率**：会员消费占比

## 常见分析场景

### 1. 销售异常分析
当某门店/区域销售额异常下降时，按以下维度下钻：
1. **时间维度**：周几、时段、天气
2. **品类维度**：哪些品类下降
3. **客户维度**：新客/老客、会员/非会员
4. **外部因素**：竞对活动、节假日、天气

### 2. 促销效果分析
- 对比促销前中后的销售数据
- 计算促销ROI
- 分析促销对后续销售的影响（透支效应）

### 3. 门店健康度评估
- 综合考虑：坪效、人效、库存周转、客户满意度
- 与同类型门店横向对比

## 归因逻辑模板

### 销售下降归因
```
1. 检查是否是全品类下降还是部分品类
2. 如果全品类：检查客流是否下降
   - 客流下降：外部因素（竞对、天气、修路等）
   - 客流不变：检查转化率和客单价
3. 如果部分品类：定位具体品类，分析原因
```

### 客流下降归因
```
1. 检查周边是否有竞对开业/促销
2. 检查是否有天气/交通因素
3. 检查门店自身是否有问题（装修、缺货等）
4. 检查营销活动是否减少
```
