# 电商行业知识库

## 核心指标体系

### 流量类指标
- **UV（独立访客）**：去重后的访问用户数
- **PV（页面浏览量）**：页面被浏览的总次数
- **跳出率**：只浏览一个页面就离开的访问比例
- **平均访问时长**：总访问时长 / 访问次数

### 转化类指标
- **加购率**：加购人数 / 访问人数
- **下单转化率**：下单人数 / 访问人数
- **支付转化率**：支付人数 / 下单人数
- **漏斗转化率**：各环节转化率

### 销售类指标
- **GMV**：成交总额
- **实际收入**：GMV - 退款 - 优惠券
- **客单价**：GMV / 订单数
- **UV价值**：GMV / UV

### 商品类指标
- **动销率**：有销售的SKU数 / 总SKU数
- **售罄率**：已售数量 / 可售库存
- **退货率**：退货订单数 / 总订单数

### 客户类指标
- **新客获取成本（CAC）**：营销费用 / 新客数
- **客户生命周期价值（LTV）**：客户在整个生命周期内的总消费
- **LTV/CAC**：健康度指标，理想值 > 3

## 大促分析框架

### 大促前（蓄水期）
- 关注：加购量、收藏量、预售订单
- 分析：流量来源、人群画像

### 大促中（爆发期）
- 关注：实时GMV、转化率、库存
- 分析：爆款表现、流量效率

### 大促后（复盘期）
- 关注：退货率、复购率、ROI
- 分析：活动效果、透支效应

## 常见分析场景

### 1. 流量分析
- 流量来源拆分（自然搜索、付费推广、直播、短视频）
- 各渠道转化率对比
- 流量成本分析

### 2. 商品分析
- ABC分类法（A类商品贡献80%销售额）
- 关联销售分析（购物篮分析）
- 价格弹性分析

### 3. 用户分析
- RFM模型（最近购买、购买频率、购买金额）
- 用户分层（新客、活跃客、沉睡客、流失客）
- 用户路径分析

## 归因逻辑模板

### 转化率下降归因
```
1. 检查是哪个环节的转化率下降
   - 流量质量：UV不变但加购率下降
   - 商品问题：加购正常但下单率下降
   - 价格问题：下单正常但支付率下降
2. 定位具体商品/页面
3. 检查是否有竞对价格战
```

### GMV下降归因
```
1. 拆分：GMV = 流量 × 转化率 × 客单价
2. 定位哪个因子下降
3. 对下降因子进一步下钻
```
