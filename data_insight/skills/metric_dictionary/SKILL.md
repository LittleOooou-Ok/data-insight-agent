# 指标字典

## 使用方法
当用户提到不熟悉的业务术语时，使用此技能查询指标定义。

## 常见指标别名映射

| 用户说法 | 标准指标名 | 计算公式 |
|---------|-----------|---------|
| 销售额/营收/流水/GMV | GMV | SUM(订单金额) |
| 成交额/实收 | 净销售额 | GMV - 退款 - 折扣 |
| 客单价/人均消费 | 客单价 | GMV / 订单数 |
| 转化率/成交率 | 转化率 | 成交人数 / 访问人数 |
| 复购率/回头率 | 复购率 | 重复购买客户数 / 总客户数 |
| 坪效/平米产出 | 坪效 | 销售额 / 门店面积 |
| 人效 | 人效 | 销售额 / 员工数 |
| 库存周转/周转率 | 库存周转率 | 销售成本 / 平均库存 |
| UV/访客/流量 | UV | 独立访客数 |
| PV/浏览量 | PV | 页面浏览次数 |

## 时间范围解读

| 用户说法 | 标准时间范围 |
|---------|-------------|
| 上个月 | 上个自然月（1日-月末） |
| 本月 | 当前自然月（1日-今天） |
| 上周 | 上个自然周（周一-周日） |
| 本季度 | 当前季度 |
| 去年同期 | 去年同月/同季度 |
| 近30天 | 今天往前推30天 |
| 近7天 | 今天往前推7天 |

## 区域维度映射

| 用户说法 | 数据库字段 |
|---------|-----------|
| 华东/华东区 | region = '华东' |
| 华北/华北区 | region = '华北' |
| 一线城市 | city_tier = 1 |
| 线上/线上渠道 | channel = 'online' |
| 线下/门店 | channel = 'offline' |

## 使用示例

当用户问"上个月华东区GMV怎么样"时：
- 上个月 → 2026年6月
- 华东区 → region = '华东'
- GMV → SUM(order_amount) WHERE region='华东' AND month='2026-06'
